Présentation de la data science


Commençons par une courte introduction


Tout d'abord, retrouvez plus d'informations dans le site : https://www.w3schools.com/python/python_ml_getting_started.asp !

Le Machine learning permet à l'ordinateur d'apprendre à partir de l'étude de données et de statistiques. C'est un pas vers l'intelligence artificielle. La data science permet d'analyser les données et d'apprendre à prédire les résultats.
Un dataset est un fichier où des informations sont stockées (ils peuvet donc être au format CSV). C'est la base de ce que nous allons analyser.
On utilisera les modules pandas et matplotlib.
